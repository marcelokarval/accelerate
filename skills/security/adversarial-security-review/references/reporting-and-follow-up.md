# Reporting And Follow-Up

Use this reference when the review finds a real seam or when it finds none.

## Minimum Finding Shape

- asset
- trust boundary
- attack class
- evidence
- impact
- exploitability assumptions
- remediation or next step

## Minimum No-Finding Shape

- attack classes checked
- evidence reviewed
- why the major classes were considered covered
- residual risks that remain out of scope

## Workflow-Level Escalation

Promote the result into workflow hardening when:

- the same failure pattern appears across multiple surfaces
- the root cause is a missing review lens or missing branch trigger
- the bug is not just local implementation drift
