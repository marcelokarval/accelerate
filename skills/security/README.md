# Security Skills

Security skills govern misuse resistance, ownership safety, auth boundaries,
secret handling, ingress posture, and fail-closed behavior.

Current skills:

| Skill | Role |
| --- | --- |
| `anti-abuse-review` | Review sensitive user-driven flows for spam, replay, enumeration, race, and availability abuse. |
| `security-patterns` | Apply the baseline for ownership, auth, secrets, and race-safe sensitive operations. |
| `adversarial-security-review` | Evidence-first hostile-path review, attack-class selection, variant hunting, and remediation routing. |
| `untrusted-ingress-hardening` | Upload/import/parser/media ingress hardening across validation, transformation, storage, and serving. |
