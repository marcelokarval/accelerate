# Data Skills

Data skills govern schema design, PostgreSQL-specific decisions, query-shape
proof, indexing, constraints, and migration-safe data evolution.

Current skills:

| Skill | Role |
| --- | --- |
| `database-design` | Relational entity, relationship, invariant, and schema-shape guidance. |
| `database-architect` | Cross-domain data architecture, lifecycle, retention, partitioning, and storage strategy. |
| `postgresql` | PostgreSQL-specific table, column, constraint, index, and feature guidance. |
| `postgres-best-practices` | PostgreSQL access-path, indexing, concurrency, migration, and operational posture. |
| `sql-optimization-patterns` | Slow-query, execution-plan, ORM shape, and index-backed optimization discipline. |
| `financial-source-truth` | Financial source-of-truth guidance for balances, charges, refunds, ledgers, reconciliation, and projections. |
| `payment-integration` | Provider-agnostic payment workflow guidance for checkout, subscriptions, idempotency, refunds, and provider events. |
| `stripe-integration` | Stripe-specific billing, webhook, subscription, refund, invoice, and provider-state guidance. |
