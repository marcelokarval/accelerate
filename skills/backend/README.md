Backend skills govern stack-profile implementation patterns, service-layer
ownership, server-driven UI contracts, and test discipline.

Current skills:

| Skill | Role |
| --- | --- |
| `django-pro` | Broad Django framework, ORM, middleware, admin, async, and structure guidance. |
| `django-service-patterns` | Django service-layer, ownership, transaction, and query-shape discipline. |
| `django-inertia-integration` | Django-side Inertia render, props, redirects, CSRF, and page-name contracts. |
| `django-vite-integration` | Django-Vite dev/prod asset pipeline, manifest, and blank-page debugging discipline. |
| `inertia-patterns` | Inertia page, layout, navigation, forms, partial reload, and server-driven prop patterns. |
| `inertia-shared-props` | Shared Inertia prop middleware and frontend PageProps contract. |
| `python-pro` | Python 3.12+ typing, structure, tooling, and maintainability baseline. |
| `python-testing` | Python/pytest behavior and regression testing discipline. |
| `celery-tasks` | Celery queue, retry, idempotency, and task-to-service delegation discipline. |
