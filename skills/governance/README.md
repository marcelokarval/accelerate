# Governance Skills

Governance skills audit architectural, stack, dependency, and policy adherence.

Current skills:

| Skill | Role |
| --- | --- |
| `governance-audit` | Audit stack and architecture adherence. |
| `api-surface-governance` | Govern runtime surface, REST/webhook/callback, and future API boundary decisions. |
| `dependency-governance` | Classify dependency legitimacy, overlap, legacy precedent, and add/remove decisions. |
| `validation-governance` | Keep backend validation authoritative and frontend validation bounded as UX convenience. |
| `codex-skill-promotion-protocol` | Repo-first skill promotion, reconciliation, synchronization, and parity validation. |
| `skill-developer` | Accelerate skill creation, adaptation, registration, synchronization, and parity verification. |
| `writing-skills` | Methodology for writing discoverable, bounded, progressively disclosed, and verifiable skills. |
