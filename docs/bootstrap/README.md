# Bootstrap Historical Provenance

This directory preserves the bootstrap context that existed before `accelerate`
became the standalone source of truth.

These files are historical provenance only. They are not current authority when
they conflict with:

- `AGENTS.md`
- `README.md`
- `SKILL.md`
- `core/`
- `adapters/`
- `profiles/`
- `onboarding/`
- `planning/`
- `skills/`
- `references/`

External paths mentioned here document origin and migration history. They must
not be used as governed Accelerate inputs unless the material has first been
imported, adapted, registered, and enforced inside this repository.
