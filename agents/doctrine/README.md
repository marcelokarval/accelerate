# Agent Doctrine

Reserved for the standalone rehoming of agent ontology, capability, governance,
and bounded execution doctrine.

In the current `standalone pre-agents` phase, this sublayer should be read as:

- the native landing zone for agent doctrine
- the place where imported `references/codex-agents/` material will be
  re-expressed
- architectural truth about bounded agents, not proof of a live agent catalog

The doctrine sublayer should eventually make explicit:

- what a bounded agent is allowed to own
- what always remains root-owned
- how family fit is judged
- how gaps are detected without forcing promotion
