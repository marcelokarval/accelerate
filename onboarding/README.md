# Onboarding

Onboarding is a first-class layer of the standalone platform.

Its job is to help `accelerate` land in a repository with little or no prior
setup by discovering:

- workflow backend
- stack profile
- runtime adapters
- docs posture
- recommended skills
- future agent candidates

The conceptual model already exists in
`docs/architecture/accelerate-onboarding-model.md`. This directory is the shell
for making that model operational.
