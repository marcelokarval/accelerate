# Onboarding Question Set

## Purpose

This document defines the minimum A&Q set for the standalone pre-agents phase.

## Ask Only When It Changes High-Impact Choices

The onboarding flow should ask the minimum number of questions needed to decide:

- workflow backend
- stack profile
- runtime wrappers
- docs posture
- current desire for bounded agents now vs later

## Minimum Question Set

1. What issue/workflow backend should be the project default?
2. What is the dominant stack profile today?
3. What runtime wrappers should be preferred?
4. What docs/publication posture exists today?
5. Should agent usage stay disabled/optional for now?
