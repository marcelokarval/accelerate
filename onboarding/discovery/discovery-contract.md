# Discovery Contract

## Purpose

This document defines what discovery must inspect before A&Q fills the gaps.

## Discovery Sources

Inspect, when present:

- repository layout
- package/runtime manifests
- CI config
- docs structure
- existing workflow tooling hints
- existing stack-specific files

## Output

Discovery should leave:

- inferred workflow backend
- inferred stack profile
- inferred runtime adapters
- inferred docs posture
- open ambiguities for A&Q
