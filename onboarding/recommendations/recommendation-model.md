# Recommendation Model

## Purpose

This document defines how onboarding should emit recommendations in the
pre-agents phase.

## Recommendation Types

The onboarding layer may recommend:

- workflow adapter
- stack profile
- runtime adapters
- docs adapter
- companion skills
- future agent candidates

## Important Rule

Recommendation is not installation or promotion.

That distinction must remain explicit for:

- skills
- adapters
- future agents
