# Core

This directory is the long-term home of the non-negotiable laws of the
standalone `accelerate` platform.

The core owns:

- root orchestration
- prompt hardening
- issue topology
- lane governance
- risk enforcement
- root closure mode
- runtime packet contracts

During the transition, the imported doctrine still lives mainly in
`references/`. Rehome material into `core/` only through deliberate bounded
slices that preserve traceability.
