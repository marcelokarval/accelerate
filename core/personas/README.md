# Personas

This layer is the native core home of persona activation, mandatory-skill
exposure, and the executive persona matrix for the standalone `accelerate`
platform.

Use this layer when the question is not only which branch is active, but which
operating posture, reviewer, or steward must be visibly in force.

Native authority now starts here:

- `model.md`
- `mandatory-skills.md`
- `executive-matrix.md`

Supporting inherited doctrine may still exist in `references/`, but persona
selection and display should no longer rely on those modules as the first stop.
