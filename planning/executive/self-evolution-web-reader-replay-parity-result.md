## Root Arbitration

Both judges converge.

Decision:

- `autoresearch / self-evolution`: `local at parity`

Bounded scope:

- external artifact intake
- source/document-backed workflow learning
- external skill vetting
- insertion-target decision
- skill-evaluation lab packet
- durable result registration
- conservative parity judgment

Excluded from this promotion:

- `local ahead` for the broad self-evolution surface
- scheduled observer runtime execution
- browser/UI polishing runtime proof
- blind-comparator output-quality proof
- direct import of GLM / Z.ai skills
- future `*.toml` agent behavior
- a concrete `web-content-reader` runtime adapter

## Residual Risks

The remaining risks are not blockers for `local at parity`, but they block
`local ahead`:

- blind comparison quality has not been exercised in a real evaluator run
- scheduled observer behavior remains architectural, not runtime-proven
- browser/UI polishing observer behavior has not been run against a live UI
- future governed `*.toml` agents do not exist yet
- any future web-reader-like adapter still needs its own adapter contract,
  network policy, proof policy, and approval gate

## Decision

Promote the broad `autoresearch / self-evolution` surface from `near parity` to:

- `local at parity`

Do not promote to:

- `local ahead`

The local standalone repository can now be treated as the primary authority for
document-backed external workflow learning in the current pre-agents phase.
